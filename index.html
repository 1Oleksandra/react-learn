Станеш впевненим кандидатом

Розумітимеш SDLC, Agile, ролі в команді та як реально будується продукт від ідеї до релізу

Матимеш реальну практику

Навчишся працювати з Jira, тест-дизайном, документацією та процесом QA як у компанії

Підготуєшся до співбесід

Mock interviews, CV/LinkedIn, відповіді на технічні питання: API, SQL, процеси, інструменти

Твоя дорога до оферу

фундамент → технічна база → портфоліо та співбесіди

Місяць 1 - Фундамент Manual QA

QA mindset • SDLC/STLC • Agile • Jira • Test Design • документація (test cases, checklists, bug reports)

Місяць 2 - Технічна база

SQL • API • Postman • Git • HTML/CSS/JS

Місяць 3 - Портфоліо та співбесіди

End-to-End підхід • фінальний проєкт • mock interviews (UA/EN) • CV + LinkedIn • стратегія пошуку роботи

Фінал - Вихід на ринок

Розбір вакансій • підготовка до перших співбесід • підтримка під час пошуку

3 проєкти для портфоліо

Це не “домашки”. Це оформлені кейси з документацією, які можна використовувати в подальшій роботі

Web Testing Project

Чеклисти, тест-кейси, баг-репорти, робота в Jira, тестування UI/UX та бізнес-логіки

API Testing Project

Postman, колекції, сценарії, валідація відповідей, робота з headers/tokens та документацією API

SQL + Database Project

SELECT, JOIN, фільтрація, агрегації. Перевірка даних у базі як частина E2E QA workflow

  • Практика на основі реальних робочих сценаріїв (SDLC, Agile, Jira, API, SQL)

  • Формування системного QA-мислення та розуміння ролі тестування в бізнесі
